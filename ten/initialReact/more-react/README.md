# basic-react

To run this project:

Clone the repo:

```js
git clone https://github.com/foundersandcoders/basic-react.git
```

Install the dependencies.
```js
npm install
```

Run the dev server.
```
npm run dev
```

Point your browser to :

```js
http://localhost:8080/
```

Make some changes to the components.js file and watch the page update in the browser
